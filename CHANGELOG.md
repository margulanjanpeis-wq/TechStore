# Changelog

Жобаның барлық маңызды өзгерістері осы файлда құжатталады.

Формат [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) негізінде,
және жоба [Semantic Versioning](https://semver.org/spec/v2.0.0.html) қолданады.

## [1.0.0] - 2024-04-28

### Қосылды
- Негізгі жобаның құрылымы
- Backend API (FastAPI)
  - Пайдаланушы авторизациясы (JWT)
  - Тауарлар CRUD операциялары
  - Себет функционалы
  - Тапсырыс жүйесі
  - Prometheus метрикалары
- Frontend (React + Vite)
  - Басты бет
  - Тауарлар каталогы
  - Тауар деталдары беті
  - Себет беті
  - Авторизация беттері
  - Тапсырыстар беті
- Database (PostgreSQL)
  - Барлық қажетті кестелер
  - Индекстер
  - Тестілік деректер
- Docker конфигурациясы
  - docker-compose.yml
  - Барлық сервистер үшін Dockerfile
- Nginx reverse proxy
  - HTTP конфигурациясы
  - HTTPS дайындығы
- Мониторинг жүйесі
  - Prometheus
  - Grafana
  - Dashboards конфигурациясы
- Terraform конфигурациясы
  - Infrastructure as Code
  - Docker ресурстары
- Скрипттер
  - Орнату скрипті
  - Backup скрипті
  - Restore скрипті
- Құжаттама
  - README.md
  - ARCHITECTURE.md
  - DEPLOYMENT.md
  - TESTING.md
  - CONTRIBUTING.md
